# Security Review

## Description
Identify security vulnerabilities and provide fixes.

## Prompt

```
Identify security risks in this snippet (inputs, auth, secrets, data handling). Provide fixes with specific patterns to apply. Keep under 170 words. Code: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Code to review

## Use Case
Catch security issues before they reach production.

Coding
