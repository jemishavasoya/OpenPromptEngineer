# Code Review Checklist

## Description
Generate comprehensive code review checklist from diff summary.

## Prompt

```
Given a diff summary, create a code review checklist: correctness, readability, edge cases, security, performance, and tests. Keep under 140 words. Diff: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Your code diff or summary

## Use Case
Ensure thorough code reviews with systematic checklist.
