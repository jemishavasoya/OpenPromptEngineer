# Commit Message Helper

## Description
Generate clear, informative commit messages.

## Prompt

```
Turn these changes into great commit messages using the format: title in imperative, body with what/why, and risks. Provide 2 options. Changes: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Description of changes

## Use Case
Write better commit messages for clearer git history.
