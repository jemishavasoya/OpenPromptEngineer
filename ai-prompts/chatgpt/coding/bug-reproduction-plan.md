# Bug Reproduction Plan

## Description
Create detailed steps to reproduce bugs consistently.

## Prompt

```
Draft steps to reproduce a bug from this report. Include environment, setup, exact actions, expected vs actual, and logging to collect. Keep under 140 words. Report: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Bug report details

## Use Case
Standardize bug reporting for faster resolution.
