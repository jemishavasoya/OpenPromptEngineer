# Regex Explainer

## Description
Explain regex patterns in plain language with examples.

## Prompt

```
Explain this regex in plain language, including what it matches, examples, and pitfalls. Keep under 140 words. Regex: {{pattern}}
```

## Variables to Replace
- `{{pattern}}` - Regex pattern to explain

## Use Case
Understand and document complex regex patterns.
