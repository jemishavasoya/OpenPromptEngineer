## 🚀 How to Use

1. Browse the categories above
2. Click on a prompt that matches your need
3. Open the markdown file
4. Copy the prompt from the code block
5. Paste into ChatGPT
6. Replace `{{variables}}` with your specific context
7. Get your result!
