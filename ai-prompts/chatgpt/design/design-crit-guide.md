# Design Crit Guide

## Description
Provide structured design critiques with actionable feedback.

## Prompt

```
Provide a design critique using: goal clarity, hierarchy, spacing, alignment, contrast, and accessibility. Give 5 actionable improvements. Keep under 170 words. Design summary: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Design description or screenshot context

## Use Case
Get systematic design feedback for improvements.
