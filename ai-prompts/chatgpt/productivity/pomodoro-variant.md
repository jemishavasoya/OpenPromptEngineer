# Pomodoro Variant

## Description
Design a customized Pomodoro-style focus routine for specific tasks.

## Prompt

```
Design a 3-cycle Pomodoro variant for {{task}}: focus length, break length, focus target, and small reward. Keep to 70 words.
```

## Variables to Replace
- `{{task}}` - The task you want to work on

## Use Case
Adapt the Pomodoro technique to your specific work style and task.
