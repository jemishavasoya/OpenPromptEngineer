# Weekly Review Script

## Description
Conduct a comprehensive weekly review to reflect and plan ahead.

## Prompt

```
Write a 20-minute weekly review for {{role}}. Sections: wins, misses, surprises, blockers, decisions made, decisions pending, top 5 for next week. Include 3 reflection questions. Keep it under 130 words.
```

## Variables to Replace
- `{{role}}` - Your job role or position

## Use Case
Maintain momentum and clarity through regular weekly reflections.

Productivity
