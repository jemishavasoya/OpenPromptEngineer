# Daily Outcome Planner

## Description
Create a structured 1-day plan with time-blocked focus sessions, breaks, and recovery periods.

## Prompt

```
Create a 1-day plan for {{role}}. Ask for my top 3 outcomes, then build a time-blocked schedule with 50-minute focus blocks, 10-minute breaks, one buffer, and one recovery block. Keep to 120 words and return as a checklist.
```

## Variables to Replace
- `{{role}}` - Your job role or position (e.g., "Product Manager", "Software Engineer")

## Use Case
Perfect for planning productive workdays with structured focus time and adequate breaks.
