# Priority Scorecard

## Description
Score and prioritize tasks based on impact, effort, urgency, and risk factors.

## Prompt

```
Give me a quick priority score for a list of tasks using impact (1–5), effort (1–5), urgency (1–5), and risk (1–5). Return a table sorted by highest score with a one-line action recommendation per task. Tasks: {{list}}
```

## Variables to Replace
- `{{list}}` - Your list of tasks to prioritize

## Use Case
Quickly determine which tasks to tackle first based on multiple factors.
