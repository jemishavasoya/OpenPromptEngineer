# Estimate Calibration

## Description
Improve estimation accuracy by learning from past estimates vs actuals.

## Prompt

```
Given tasks and past estimates vs actuals, recalibrate. Return a table: Task, Past Estimate, Actual, New Estimate, Risk Note. Keep within 100 words. Data: {{details}}
```

## Variables to Replace
- `{{details}}` - Your historical estimation data

## Use Case
Build better estimation skills through data-driven calibration.

Productivity
