# Habit Tracker Snapshot

## Description
Design a simple 7-day habit tracking system with triggers and rewards.

## Prompt

```
Design a 7-day habit tracker for {{habit}} with a daily 2-minute reflection question, a trigger, and a reward. Keep under 90 words.
```

## Variables to Replace
- `{{habit}}` - The habit you want to build

## Use Case
Build new habits with a structured tracking and reward system.
