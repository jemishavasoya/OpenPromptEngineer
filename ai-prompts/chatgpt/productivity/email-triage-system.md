# Email Triage System

## Description
Create a systematic approach to managing email inbox with clear rules and templates.

## Prompt

```
Design a 3-bucket email triage rule set (Do now, Delegate, Defer) for {{role}}. Give criteria, canned subject tags, and a 2-sentence reply template for each bucket. Keep it under 120 words.
```

## Variables to Replace
- `{{role}}` - Your job role or position

## Use Case
Streamline email management and reduce inbox overwhelm.

Productivity
