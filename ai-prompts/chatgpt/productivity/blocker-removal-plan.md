# Blocker Removal Plan

## Description
Generate quick solutions to unblock stuck tasks.

## Prompt

```
Ask for my blocker, then propose 3 ways to unblock in under 30 minutes, plus 1 longer-term fix. Keep to 80 words. Blocker: {{issue}}
```

## Variables to Replace
- `{{issue}}` - Description of what's blocking you

## Use Case
Overcome obstacles quickly with multiple solution paths.

## Category
Productivity
