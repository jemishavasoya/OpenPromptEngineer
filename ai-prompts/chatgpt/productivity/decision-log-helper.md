# Decision Log Helper

## Description
Document decisions systematically for future reference.

## Prompt

```
Convert this decision into a log entry: context, options considered, chosen option, rationale, date, owner, and review date. Keep to 90 words. Decision: {{details}}
```

## Variables to Replace
- `{{details}}` - Description of the decision made

## Use Case
Maintain a clear record of important decisions and their reasoning.

## Category
Productivity
