# Workback Plan

## Description
Create a reverse timeline from deadline to present with milestones.

## Prompt

```
Create a workback plan from deadline {{date}} for {{deliverable}}. Include milestones, owners, risks, and buffers. Keep to 120 words and deliver as bullets.
```

## Variables to Replace
- `{{date}}` - Your deadline date
- `{{deliverable}}` - What needs to be delivered

## Use Case
Plan complex projects by working backwards from the deadline.
