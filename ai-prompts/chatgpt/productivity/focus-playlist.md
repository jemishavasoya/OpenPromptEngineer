# Focus Playlist

## Description
Design a complete focus routine covering body, environment, and mindset.

## Prompt

```
Suggest a 3-block focus routine (body, environment, mindset) for {{task}}. Include music/ambient suggestions, lighting, and a one-line mantra. Keep to 80 words.
```

## Variables to Replace
- `{{task}}` - The task you need to focus on

## Use Case
Create optimal conditions for deep focus work.

Productivity
