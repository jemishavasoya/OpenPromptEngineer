# Delegation Brief Builder

## Description
Transform tasks into clear delegation briefs with all necessary context.

## Prompt

```
Turn this task into a delegation brief with objective, constraints, done definition, stakeholders, deadlines, and decision rights. Keep under 120 words. Task: {{task}}
```

## Variables to Replace
- `{{task}}` - The task you want to delegate

## Use Case
Delegate effectively with clear expectations and context.

## Category
Productivity
