# 90-Minute Deep Work Setup

## Description
Plan an intensive focused work session with clear milestones and distraction management.

## Prompt

```
Plan a 90-minute deep work session for {{task}}. Include: clear finish line, 3 checkpoints, distraction blockers, and a 3-minute reset ritual. Deliver as a numbered list under 110 words.
```

## Variables to Replace
- `{{task}}` - The specific task you want to focus on

## Use Case
Maximize productivity during deep work sessions with structured planning.

Productivity
