# Standup Sharpening

## Description
Create crisp, outcome-focused standup updates.

## Prompt

```
Craft a standup update that is crisp and outcome-based. Format: Yesterday (outcomes), Today (one measurable goal), Risks (with mitigation), Help needed. Keep under 70 words. Context: {{notes}}
```

## Variables to Replace
- `{{notes}}` - Your work context and notes

## Use Case
Deliver clear, actionable standup updates that respect everyone's time.
