# SWOT Snapshot

## Description
Conduct SWOT analysis for strategic planning.

## Prompt

```
Create a SWOT snapshot for {{company}} with 3 bullets per quadrant and a one-line implication. Keep under 170 words.
```

## Variables to Replace
- `{{company}}` - Company or product name

## Use Case
Assess strategic position and opportunities.

## Category
Operations & Strategy
