# Microcopy Polish

## Description
Refine UI microcopy for clarity and brevity.

## Prompt

```
Polish this microcopy for clarity and brevity. Return variants for button (<=14 chars), helper text (<=18 words), and tooltip (<=14 words). Copy: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Your microcopy text

## Use Case
Improve user interface text for better UX.

Writing
