# Style Guide Snapshot

## Description
Create mini style guides for consistent brand voice.

## Prompt

```
Summarize a mini style guide: tone words, banned phrases, preferred structures, and examples for openings/closings. Keep under 140 words. Brand: {{details}}
```

## Variables to Replace
- `{{details}}` - Brand information and preferences

## Use Case
Maintain consistent brand voice across content.

Writing
