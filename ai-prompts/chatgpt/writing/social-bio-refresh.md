# Social Bio Refresh

## Description
Create multiple bio variations for social media profiles.

## Prompt

```
Write 3 bio options for {{platform}}. Formats: authoritative, friendly, crisp. Max 140 characters each. Include 1 proof point and 1 CTA link placeholder.
```

## Variables to Replace
- `{{platform}}` - The social media platform (Twitter, LinkedIn, etc.)

## Use Case
Optimize social media profiles for different tones.

Writing
