# CTA Upgrader

## Description
Create compelling calls-to-action using different psychological triggers.

## Prompt

```
Suggest 5 strong calls-to-action for {{page}}. Include one urgency angle, one value-first, one social proof, one risk-reversal, one curiosity. Keep each under 10 words.
```

## Variables to Replace
- `{{page}}` - The page or context for the CTA

## Use Case
Improve conversion rates with better CTAs.
