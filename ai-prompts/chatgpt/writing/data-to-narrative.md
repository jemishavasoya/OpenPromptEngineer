# Data-to-Narrative

## Description
Transform raw data into compelling narrative with clear takeaways.

## Prompt

```
Turn these data points into a narrative paragraph with one clear takeaway and one action. Keep to 90 words. Data: {{paste}}
```

## Variables to Replace
- `{{paste}}` - Your data points or statistics

## Use Case
Make data more accessible and actionable through storytelling.
