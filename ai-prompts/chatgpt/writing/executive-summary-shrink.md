# Executive Summary Shrink

## Description
Condense documents into concise executive summaries.

## Prompt

```
Condense this document into an executive summary: goal, key facts, decision needed, risks, next step. Limit to 120 words. Content: {{paste}}
```

## Variables to Replace
- `{{paste}}` - The document content to summarize

## Use Case
Create executive-level summaries for busy stakeholders.
