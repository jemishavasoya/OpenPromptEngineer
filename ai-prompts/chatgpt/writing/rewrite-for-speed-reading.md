# Rewrite for Speed Reading

## Description
Format text for easy scanning and quick comprehension.

## Prompt

```
Rewrite the text with line breaks after each sentence, bold the verbs, and keep sentences under 14 words. Limit to 120 words. Text: {{paste}}
```

## Variables to Replace
- `{{paste}}` - The text to reformat

## Use Case
Make content more scannable for busy readers.
