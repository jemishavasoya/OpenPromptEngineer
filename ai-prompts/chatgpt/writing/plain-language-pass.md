# Plain Language Pass

## Description
Simplify complex text to 6th-grade reading level without losing accuracy.

## Prompt

```
Rewrite the text to a 6th-grade reading level without losing accuracy. Use short sentences and everyday words. Keep under 120 words. Text: {{paste}}
```

## Variables to Replace
- `{{paste}}` - The text to simplify

## Use Case
Make content accessible to broader audiences.

## Category
Writing
