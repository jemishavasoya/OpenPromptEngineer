# Proofreading with Rationale

## Description
Get detailed proofreading with explanations for each correction.

## Prompt

```
Proofread the text and list fixes with reasons. Return as: Original, Fix, Rationale. Keep under 120 words. Text: {{paste}}
```

## Variables to Replace
- `{{paste}}` - The text to proofread

## Use Case
Learn from corrections to improve writing skills.

Writing
