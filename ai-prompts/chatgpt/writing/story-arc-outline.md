# Story Arc Outline

## Description
Structure content using a classic 5-beat story arc.

## Prompt

```
Create a 5-beat story arc for {{topic}} using Setup, Tension, Turning Point, Resolution, CTA. Return as bullet outline under 120 words.
```

## Variables to Replace
- `{{topic}}` - Your story topic or subject

## Use Case
Create compelling narratives for presentations, marketing, or content.
