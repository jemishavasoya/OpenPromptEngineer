# Battlecard Snapshot

## Description
Create competitive battlecards for sales team.

## Prompt

```
Create a battlecard for competitor {{name}}: summary, 3 strengths, 3 counters, proof, trap questions, and landmines to avoid. Keep under 180 words.
```

## Variables to Replace
- `{{name}}` - Competitor name

## Use Case
Equip sales team to compete effectively.

## Category
Marketing & Sales
