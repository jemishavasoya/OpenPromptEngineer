# Mutual Action Plan

## Description
Create shared accountability plan with prospects.

## Prompt

```
Draft a mutual action plan: milestones, owners (us/them), due dates, and risks. Add a one-line success definition. Keep under 150 words. Deal: {{details}}
```

## Variables to Replace
- `{{details}}` - Deal context

## Use Case
Align on next steps and close deals faster.

Marketing & Sales
