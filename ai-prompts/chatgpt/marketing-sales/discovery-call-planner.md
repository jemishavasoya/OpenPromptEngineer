# Discovery Call Planner

## Description
Structure effective discovery calls with strategic questions.

## Prompt

```
Create a discovery call plan for {{prospect}}. Include opener, 6 problem questions, 3 impact questions, budget/timeline checks, and close with next step. Keep under 160 words.
```

## Variables to Replace
- `{{prospect}}` - Prospect information

## Use Case
Run discovery calls that uncover real needs.

Marketing & Sales
