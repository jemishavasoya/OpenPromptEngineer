# Content Calendar

## Description
Plan monthly content calendar with variety.

## Prompt

```
Propose a 4-week content calendar for {{audience}} with 3 posts per week. Include format (video, carousel, blog), hook, and CTA. Keep under 180 words.
```

## Variables to Replace
- `{{audience}}` - Target audience

## Use Case
Maintain consistent content output with strategic planning.
