# Webinar Brief

## Description
Plan engaging webinars with clear structure.

## Prompt

```
Draft a webinar plan: topic, title, agenda, demo flow, attendee promises, follow-up offer, and 3 audience questions. Keep under 170 words. Topic: {{topic}}
```

## Variables to Replace
- `{{topic}}` - Webinar topic

## Use Case
Run webinars that educate and convert.

## Category
Marketing & Sales
