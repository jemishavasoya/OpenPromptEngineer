# ROI Snapshot

## Description
Create quick ROI calculations for prospects.

## Prompt

```
Create a quick ROI snapshot for {{product}}. Inputs: price, baseline metric, expected lift. Output: payback time, 1 proof point, and a one-line pitch. Keep under 130 words.
```

## Variables to Replace
- `{{product}}` - Your product

## Use Case
Quantify value proposition for prospects.

## Category
Marketing & Sales
