# Demo Script

## Description
Structure product demos that connect features to pains.

## Prompt

```
Write a 10-minute demo script for {{product}}. Sections: set agenda, 3 pains, 3 feature moments tied to pains, proof point, next step. Keep under 170 words.
```

## Variables to Replace
- `{{product}}` - Your product

## Use Case
Deliver demos that resonate with prospects.

## Category
Marketing & Sales
